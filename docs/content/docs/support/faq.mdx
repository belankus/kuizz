# Frequently Asked Questions

## General

### What is Kuizz?

Kuizz is a real-time interactive quiz platform for live events, classrooms, team training, and entertainment. It supports multiple question types, real-time gameplay with leaderboards, and detailed analytics.

### Is Kuizz free to use?

Yes! Kuizz offers a free plan that includes:
- Unlimited quiz creation
- Up to 50 players per session
- 30-day analytics retention
- Basic features

Paid plans unlock higher player limits, longer data retention, API access, and webhooks.

### What devices do participants need?

Participants only need a modern web browser — mobile, tablet, or desktop. No app download is required. Players join by visiting `kuizz.my.id/join` and entering the game PIN.

## Quizzes & Questions

### How many questions can a quiz have?

There is no limit on the number of questions per quiz.

### Can I reuse questions across quizzes?

Not directly yet — this feature is planned for a future release. For now, you can duplicate an entire quiz and modify it.

### Can participants see the correct answer after answering?

Yes. You can enable **Show Correct Answer** in your quiz settings. When enabled, participants see the correct answer immediately after each question.

### How is the scoring calculated?

Points are based on:
- **Correctness** — only correct answers earn points
- **Speed** — faster answers earn a speed bonus of up to 500 points
- **Streaks** — 3+ consecutive correct answers activate a 1.5× multiplier

## Sessions & Gameplay

### How many participants can join a session?

| Plan | Max Players |
|------|-------------|
| Free | 50 |
| Pro | 500 |
| Enterprise | Unlimited |

### What happens if a player disconnects mid-game?

Players can rejoin a session using the same game PIN. Their progress and score are preserved for up to 5 minutes after disconnecting.

### Can I pause a live session?

Yes. The host can pause the current question timer at any time from the host screen.

## Data & Privacy

### Where is my data stored?

All data is stored on servers in Indonesia. We are GDPR-compliant and follow global data protection best practices.

### How do I delete my account?

Go to **Dashboard → Settings → Account → Delete Account**. This permanently deletes all your quizzes, sessions, and analytics data.

<Callout variant="warning">
  Account deletion is **permanent and irreversible**. Export your data before deleting.
</Callout>

### Can I export my quiz data?

Yes. Go to any quiz and click **Export → Download JSON** to download the full quiz data.
