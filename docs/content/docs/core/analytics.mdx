# Analytics

Kuizz provides detailed analytics for every quiz session, helping you understand participant performance and quiz effectiveness.

## Session Reports

After every live session ends, a full report is automatically generated and accessible from **Dashboard → Sessions**.

### Report Overview

Each session report includes:

| Metric | Description |
|--------|-------------|
| **Total Players** | Number of participants who joined |
| **Completion Rate** | % of players who finished all questions |
| **Average Score** | Mean score across all participants |
| **Average Accuracy** | % of correct answers across all players |
| **Session Duration** | Total time from start to finish |

## Question-Level Analytics

For each question in the session, you can see:

- **Response distribution** — how many players chose each answer
- **Accuracy rate** — percentage who answered correctly
- **Average response time** — how quickly players answered
- **Skip rate** — percentage who didn't answer in time

```json title="Example Question Analytics Response"
{
  "questionId": "q_abc123",
  "question": "What is the capital of Indonesia?",
  "totalResponses": 42,
  "correctResponses": 36,
  "accuracyRate": 85.7,
  "avgResponseTime": 8.3,
  "distribution": {
    "Jakarta": 36,
    "Surabaya": 4,
    "Bandung": 2,
    "Medan": 0
  }
}
```

## Player Leaderboard

The leaderboard ranks participants by score. Points are awarded based on:

1. **Correctness** — base points for a correct answer (default: 1000)
2. **Speed bonus** — faster answers earn more points
3. **Streak bonus** — consecutive correct answers multiply points

```ts title="Scoring Formula"
const speedBonus = Math.floor((timeLimit - responseTime) / timeLimit * 500);
const streakMultiplier = streak >= 3 ? 1.5 : 1;
const totalPoints = (basePoints + speedBonus) * streakMultiplier;
```

## Exporting Data

Session data can be exported in two formats:

<Tabs items={['CSV', 'JSON']}>
  <Tab value="CSV">
    Download a spreadsheet-compatible file with one row per player, including their score, accuracy, and per-question responses.

    Go to **Session Report → Export → Download CSV**.
  </Tab>
  <Tab value="JSON">
    Download the full raw session data including all responses, timestamps, and metadata.

    ```bash
    GET /api/sessions/{sessionId}/export?format=json
    Authorization: Bearer {token}
    ```
  </Tab>
</Tabs>

## Analytics API

You can fetch analytics programmatically:

```bash
# Get session summary
GET /api/sessions/{sessionId}/analytics

# Get question-level breakdown
GET /api/sessions/{sessionId}/analytics/questions

# Get player leaderboard
GET /api/sessions/{sessionId}/leaderboard
```

See [API Authentication](/docs/developer/authentication) for how to authenticate requests.

<Callout variant="tip" title="Data Retention">
  Session analytics are retained for **90 days** on the free plan and **indefinitely** on paid plans.
</Callout>
