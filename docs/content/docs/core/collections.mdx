# Collections

Collections let you group related quizzes together — useful for organizing quizzes by topic, course, event, or team.

## What Is a Collection?

A collection is a named set of quizzes. Think of it as a **folder** for your quiz library.

Examples:
- `📚 Math Grade 7` — all quizzes for a specific class
- `🎉 Company Onboarding` — quiz set for new employee training
- `🏆 Weekly Trivia` — recurring game night quizzes

## Creating a Collection

<Steps>
  <Step>
    Go to **Dashboard → Collections**
  </Step>
  <Step>
    Click **New Collection**
  </Step>
  <Step>
    Enter a **name** and optional **description**
  </Step>
  <Step>
    Choose a **cover image** (optional)
  </Step>
  <Step>
    Click **Create**
  </Step>
</Steps>

## Adding Quizzes to a Collection

You can add quizzes to a collection two ways:

### From the Quiz Editor

Open any quiz and click the **Collections** tab in the settings panel. Select one or more collections to add the quiz to.

### From the Collection Page

1. Open the collection
2. Click **Add Quizzes**
3. Search and select quizzes from your library
4. Click **Add Selected**

<Callout variant="info">
  A quiz can belong to **multiple collections** at the same time.
</Callout>

## Collection Visibility

Collections share the same visibility settings as quizzes:

| Visibility | Who Can See |
|------------|-------------|
| `public` | Anyone with the link |
| `private` | Only you |
| `unlisted` | Anyone with the direct link |

## Sharing a Collection

You can share a collection via its public URL. Shared collections display:
- Collection name and description
- List of quizzes (title, question count)
- Play buttons for each quiz

```
https://kuizz.my.id/collection/{collection-slug}
```

## Collection API

Collections are fully accessible via the REST API. See the [API Overview](/docs/developer/api-overview) for details.

```bash
# List all collections
GET /api/collections

# Create a collection
POST /api/collections
Content-Type: application/json

{
  "name": "Company Onboarding",
  "description": "Quiz set for new hires"
}
```
