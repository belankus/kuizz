# Creating a Quiz

The Kuizz quiz editor provides a powerful yet simple interface for building interactive quizzes. This guide covers everything from question types to advanced settings.

## Opening the Editor

From the dashboard, click **New Quiz** or select an existing quiz to open the editor. The editor consists of:

- **Left panel** — question list and ordering
- **Center panel** — active question editor
- **Right panel** — question settings (time, points, media)

## Question Types

Kuizz supports four question types:

### Multiple Choice

The most common type. Participants choose from 2 to 4 options.

```mdx title="Multiple Choice Configuration"
- Question text (required)
- 2–4 answer options
- 1 correct answer
- Time limit: 5–120 seconds
- Points: 0–2000
```

### True / False

A binary question ideal for testing factual knowledge.

### Open Ended

Participants type a free-text answer. Uses **fuzzy matching** to auto-grade responses against an expected answer.

<Callout variant="note" title="Manual Grading">
  Open Ended questions with complex answers can be flagged for manual review in the session analytics.
</Callout>

### Slide (Information)

Not a question — a presentation slide shown to participants during the quiz for context or transitions.

## Rich Media Support

Each question supports:

| Media Type | Formats |
|-----------|---------|
| Image | JPG, PNG, GIF, WebP |
| Video | MP4, YouTube embed |
| Audio | MP3, WAV |

Images and videos appear above the question text on participant screens.

## Question Settings

| Setting | Default | Range |
|---------|---------|-------|
| Time Limit | 30s | 5–120s |
| Points | 1000 | 0–2000 |
| Point Multiplier | 1× | 1×, 2× |
| Media | None | Image / Video |

## Organizing Questions

Drag and drop questions in the left panel to reorder them. You can also:

- **Duplicate** a question by right-clicking it
- **Delete** a question using the trash icon
- **Add a separator** to group questions visually

## Quiz Settings

Access quiz-level settings via the gear icon in the top bar:

```ts title="Quiz Settings"
interface QuizSettings {
  title: string;
  description?: string;
  coverImage?: string;
  visibility: 'public' | 'private' | 'unlisted';
  randomizeQuestions: boolean;
  randomizeAnswers: boolean;
  showLeaderboard: boolean;
}
```

## Publishing

A quiz must meet the following requirements before it can be published:

- At least **1 question** added
- Every question has a **title**
- Every multiple choice question has at least **2 options** with **1 correct answer**

Click **Publish** in the top-right corner to make the quiz available for hosting live sessions.

<Callout variant="tip" title="Auto-Save">
  The editor automatically saves your work every 30 seconds. You can also press **Ctrl+S** to save manually.
</Callout>
