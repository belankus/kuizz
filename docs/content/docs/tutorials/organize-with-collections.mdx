# Tutorial: Organizing with Collections

## Overview

In this tutorial, you'll learn how to tidy up your library by creating collections and moving quizzes into them.

## How It Works

1.  **Create Collection**: Go to the "Collections" tab and click "New Collection".
2.  **Title & Categorize**: Name your collection (e.g., "Math 101") and add an optional cover image.
3.  **Populate**: Navigate back to your library, click the menu on a quiz, and select "Move to Collection".
4.  **Confirm**: View your collection to see the organized quizzes.

## Key Components

-   **Collection Management UI**: `frontend/src/components/collections/CollectionsPage.tsx`
-   **Prisma Model**: `Collection` and `CollectionItem`.

## Code References

-   **Collections Backend**: `backend/src/collections/collections.service.ts`

## Notes

Organizing content into collections is the first step toward collaborative work with other users.
