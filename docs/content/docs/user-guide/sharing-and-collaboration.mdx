# Sharing and Collaboration

## Overview

Kuizz is better with teammates. Learn how to share your collections and work together on building the best quiz content.

## How to Collaborate

1.  **Open Collection Settings**: Navigate to a collection you own.
2.  **Invite Members**: Click "Invite Member" and enter their email address.
3.  **Set Role**:
    -   **Viewer**: Can host games from this collection.
    -   **Editor**: Can add, remove, and edit quizzes.
4.  **Accept Invite**: The invited user will see the collection in their "Shared with me" tab.

## Key Components

-   **Roles**: Granular control over what collaborators can do.
-   **Invitations**: Secure way to grant access via user email.

## Code References

-   **Permissions Logic**: `backend/src/lib/permissions/collection.permissions.ts`
-   **Member Management**: `frontend/src/components/collections/CollectionHeader.tsx`

## Notes

Only the Owner of a collection can delete the collection or remove other members.
