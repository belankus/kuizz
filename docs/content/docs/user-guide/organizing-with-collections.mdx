# Organizing with Collections

## Overview

As your library grows, finding specific quizzes can become challenging. Collections allow you to group related quizzes together for better organization and easier access.

## How to Use Collections

1.  **Create a Collection**: Click the "New Collection" button in the collections tab.
2.  **Add Quizzes**:
    -   Open a quiz and select "Add to Collection".
    -   Or, from the Collection view, click "Add Quiz" and select from your library.
3.  **Customize**: Add a cover image and description to help identify the collection's purpose.

## Key Components

-   **Visibility**: Choose between PRIVATE (only you and members) or PUBLIC (anyone with the link).
-   **Cover Image**: Helps visually distinguish different sets of academic subjects or training modules.

## Code References

-   **Collections List**: `frontend/src/app/dashboard/collections/page.tsx`
-   **Detail View**: `frontend/src/app/dashboard/collections/[collectionId]/page.tsx`

## Notes

A single quiz can be part of multiple collections. Deleting a collection does not delete the quizzes inside it.
