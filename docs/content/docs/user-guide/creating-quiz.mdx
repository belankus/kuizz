# Creating a Quiz

## Overview

Learn how to create your first quiz in Kuizz. The creation process is designed to be fast and intuitive, allowing you to go from an idea to a live game in minutes.

## How to Create a Quiz

1.  **Log In**: Ensure you are logged into your Kuizz account.
2.  **Navigate to Dashboard**: Click on the "My Library" or "Dashboard" link in the sidebar.
3.  **Click "Create New"**: Find the "Create New Quiz" card (it has a large plus icon).
4.  **Enter Metadata**: Provide a catchy title and a brief description for your quiz.
5.  **Save as Draft**: Your quiz is automatically saved as a draft initially.

## Key Components

-   **Title**: The main name of your quiz displayed on the library and in the game lobby.
-   **Description**: A short summary of what the quiz covers.
-   **Cover Image**: (Optional) Add a visual touch to your quiz card.

## Code References

-   **Creation Page**: `frontend/src/app/dashboard/(quiz)/create/page.tsx`
-   **Backend Controller**: `backend/src/quiz/quiz.controller.ts`

## Notes

You don't need to add all questions at once. You can save your quiz as a draft and come back to it later from your Library.
